# YougetGUI
https://github.com/soimort/you-get 的Windows图形化界面实现

you-get 是一个支持几乎所有主流视频网站视频下载的python脚本实现



# 使用方法
1. 安装java(8u40+)版本

2. 下载最新的YogetGUI release

3. 双击运行YougetGUI.jar（或者使用java -jar YougetGUI.jar路径）

# 有什么特点

1. 提供图形化界面，显示下载进度和速度

2. 支持多个下载任务顺序下载

3. 自动保存下载记录




# 实现原理

通过调用you-get命令行来实现视频下载
